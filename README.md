# Hex

This project implements a mobile and desktop version of the hex game.

## Requirements

Actually, you need to have this tools installed to code in the project.

- Node 12
- Yarn

## Installation

```bash
make install
```

## Usage

```bash
make start
```

## Testing

```bash
make test
```
