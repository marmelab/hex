# Hex

This project implements a mobile version of the hex game.

![Board](https://user-images.githubusercontent.com/9250470/78352516-bb90b580-75a8-11ea-995b-39a97772b2e8.png)

## Requirements

- Node 12 or above
- Yarn

## Installation

```bash
make install
```

## Usage

```bash
make start
```

## Testing

```bash
make test
```
